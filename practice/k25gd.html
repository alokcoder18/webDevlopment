Text Formatting Tags in HTML

Defintion:

These tags help you emphasize, define, or style text semantically, meaning they give meaning to the text, which is also helpful for accessibility and SEO.

Different types of Text Formatting tags

Strong Tag

The Strong tag indicates strong importance or seriousness. It usually rendered as bold by browsers.

Emphasize Tag

It indicates emphasis (stress) on the text. It usually rendered as italic. The screen readers may change voice tone.

Mark Tag

It highlights or marks text (like using a highlighter pen). It usually rendered with a yellow background.

Small Tag

It represents side comments or fine print.It usually rendered as smaller text.

Abbreviation Tag

It defines an abbreviation or acronym. The full form can be provided with the title attribute, which appears on hover.

The HTML language is used for web pages.

Definition Tag

It represents the defining instance of a term (the term being defined).It is typically shown in italics. It is useful in glossaries or definitions.

Defintion of Block Level elements

Block Level Element start on a new line and take up the full width available (by default). It Can contain other block-level elements and inline elements. It create rectangular blocks in the layout.

Quatations in HTML

There are 2 methods of writing Quatations
1. Q Tag : q tag is used for short, inline quotations inside a paragraph. The browsers usually add quotation marks automatically.
2. blockquotation tag: It is used for longer quotations displayed as a separate block (usually indented). It can include multiple paragraphs. It often used to quote external sources or large chunks of text.
